Professional photogrammetry workflow for e-commerce. Studio-captured 3D models with Draco compression deployed as interactive web viewers styled to match each product's aesthetic.

Overview

Complete photogrammetry-to-web pipeline for e-commerce product visualization. Both models captured in controlled home studio using DSLR equipment, studio lighting, and motorized turntable. The workflow prioritizes high fidelity while maintaining web performance through aggressive optimization.

Rather than generic 3D viewers, I built custom mock e-commerce websites for each product—collaborating with Claude AI to design interfaces matching each product's aesthetic. Shoe viewer: sleek modern design. Clay bowl: handcrafted artisanal styling. Both self-hosted, demonstrating how 3D product visualization can be tailored to brand identity.

Capture & Processing Workflow

1. Studio Photography: Hundreds of photographs per product using DSLR on tripod, shooting each object on motorized turntable under controlled studio lighting. Multiple passes at different heights ensured complete coverage.

2. Lightroom Processing: Batch color correction and lens distortion correction applied to all images for consistent exposure and optical aberration elimination.

3. Metashape Pro Reconstruction: Imported corrected image sets into Agisoft Metashape Pro. Software performed feature matching, built dense point clouds, generated high-poly meshes, and baked texture maps. Result: photorealistic 3D models with millions of polygons and 4K+ texture resolution.

4. Blender Cleanup: Imported models into Blender for final optimization. Removed stray geometry, retopologized high-density areas, optimized UV layouts, prepared for web deployment.

5. Draco Compression: Applied Draco geometry compression achieving ~90% file size reduction while maintaining visual fidelity. Critical for web delivery—uncompressed models would be prohibitively large.

6. Web Viewer Development: Built custom interactive 3D viewers using Three.js, with UI design iterated collaboratively with Claude AI. Each site's styling reflects its product.

7. Self-Hosting: Deployed both experiences on self-hosted infrastructure, demonstrating full ownership of the 3D pipeline from capture through delivery.

E-Commerce Integration

The final deliverables aren't just 3D models—they're complete product presentation experiences. Each mock e-commerce site demonstrates how photogrammetry can integrate into online retail, allowing customers to examine products from every angle.

Design process involved multiple iterations with Claude AI, refining typography, color schemes, layout, and interaction patterns. This collaborative approach shows how AI tools can accelerate design iteration while maintaining creative direction.
